** O2C Demo Project — Angular + .NET**
This is a full-stack demo application that simulates a simple Order-to-Cash (O2C) flow built using Angular and .NET Minimal API.
It demonstrates frontend-backend integration, API communication, and basic eCommerce style workflow.

**Tech Stack**
Frontend
Angular
TypeScript
HTML / CSS
Backend
.NET Minimal API
C#
Communication
REST APIs (HTTP)

User Registration
Login
Product Catalog
Add to Cart
Checkout
Orders
Payments

**Setup Guide**
Clone Repository
git clone https://github.com/arpit-gupta10/02c-project.git
cd 02c-project


Run Backend
cd backend
dotnet run

Backend will start on localhost with a configured port.

Run Frontend
cd frontend/o2c-ui
npm install
ng serve

**Open browser:**
http://localhost:4200
